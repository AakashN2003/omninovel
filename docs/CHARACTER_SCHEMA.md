# Character Schema

## Purpose

Create consistent characters that can evolve throughout a novel.

---

## Structure

```json
{
  "identity": {
    "name": "",
    "age": 0,
    "gender": "",
    "race": "",
    "role": ""
  },

  "personality": [],

  "goals": {
    "primary": "",
    "secondary": []
  },

  "strengths": [],

  "weaknesses": [],

  "relationships": [],

  "status": {
    "alive": true,
    "location": "",
    "power_level": ""
  },

  "memory": [],

  "evolution_log": []
}
```

---

## Outputs

- Character Profile
- Relationships
- Status
- Memory
- Evolution Log
