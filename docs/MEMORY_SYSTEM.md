# Memory System

## Purpose

Maintain consistency across chapters.

---

## Memory Types

### Universe Memory

Stores:

- World Laws
- Regions
- Factions
- Power System

### Character Memory

Stores:

- Personality
- Relationships
- Important Events
- Current Status

### Story Memory

Stores:

- Current Arc
- Major Events
- Plot Progression

---

## Generation Flow

Create Universe
↓
Create Characters
↓
Generate Outline
↓
Generate Chapter
↓
Store Summary
↓
Store Important Facts
↓
Update Memory

---

## Chapter Generation

Before generating a chapter:

1. Load Universe Memory
2. Load Character Memory
3. Load Story Memory
4. Generate Chapter
5. Update Memory
