# Universe DNA Schema

## Purpose

Universe DNA is the foundation of every novel.

Every character, location, faction, event, and chapter must follow the Universe DNA.

---

## Structure

```json
{
  "name": "",
  "genre": "",
  "theme": "",
  "tone": "",

  "world_laws": {
    "magic_exists": false,
    "gods_exist": false,
    "immortality_possible": false,
    "time_travel": false
  },

  "power_system": {
    "type": "",
    "energy_source": "",
    "realms": []
  },

  "civilization_level": "",

  "regions": [],

  "factions": [],

  "races": [],

  "timeline": []
}
```

---

## Outputs

- Universe Summary
- World Laws
- Power System
- Regions
- Factions
- Timeline
